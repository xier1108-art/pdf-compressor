# PDF 압축기

인터넷 없이 오프라인에서 사용할 수 있는 가벼운 PDF 압축 도구입니다.  
[ilovepdf.com](https://www.ilovepdf.com/ko/compress_pdf) 와 동일한 방식으로 PDF 내 이미지를 재압축하여 파일 크기를 줄입니다.

---

## 주요 기능

- **오프라인 동작** – 인터넷 연결 불필요
- **3단계 압축 수준** – 최대 압축 / 권장(균형) / 저압축(화질 우선)
- **배치 처리** – 여러 PDF 파일을 한 번에 압축
- **드래그 앤 드롭** 지원 (Python 환경에서 실행 시)
- **원본 파일 보존** – 압축 결과는 별도 폴더에 저장
- **압축 결과 표시** – 원본 크기 / 압축 크기 / 압축률 한눈에 확인

## 다운로드 (바로 실행)

> [**Releases**](../../releases) 탭에서 `PDF압축기.exe` 를 다운로드하세요.

---

## 개발자 실행 방법

### 1. 의존성 설치

```bash
pip install -r requirements.txt
```

| 패키지 | 용도 |
|--------|------|
| `pymupdf` | PDF 파싱 및 이미지 재압축 (MuPDF 엔진) |
| `pikepdf` | 구조 정리, 스트림 압축 (QPDF 기반) |
| `Pillow` | 이미지 리사이즈 및 JPEG 인코딩 |
| `tkinterdnd2` | 드래그 앤 드롭 지원 |

### 2. 실행

```bash
python app.py
```

또는 `run.bat` 더블클릭

---

## 압축 수준 안내

| 모드 | 이미지 최대 크기 | JPEG 품질 | 예상 압축률 |
|------|----------------|----------|------------|
| 최대 압축 | 900px | 35% | 스캔본 기준 70~90% 감소 |
| 권장 (기본값) | 1600px | 65% | 스캔본 기준 40~70% 감소 |
| 저압축 | 2400px | 85% | 스캔본 기준 20~40% 감소 |

> 텍스트/벡터 전용 PDF는 이미지가 없어 압축 효과가 적을 수 있습니다.

## .exe 직접 빌드

```bash
build.bat
```

빌드 결과물: `dist/PDF압축기.exe`  
Python이 설치되어 있어야 합니다.

---

## 기술 스택

- **Python 3.10+**
- **PyMuPDF (fitz)** – PDF 이미지 추출 및 교체
- **pikepdf** – 구조적 압축 및 메타데이터 정리
- **Pillow** – 이미지 처리 (리사이즈, JPEG 재인코딩)
- **tkinter + tkinterdnd2** – 데스크탑 GUI
